1. Pendahuluan

1.1 Latar Belakang

Filter Band Pass (BPF) adalah filter yang meloloskan sinyal pada rentang frekuensi tertentu (passband) dan meredam sinyal di luar rentang tersebut (stopband). Filter ini banyak digunakan dalam aplikasi seperti pengolahan audio, komunikasi, dan pemrosesan sinyal biomedis.

1.2 Tujuan

Merancang filter BPF digital menggunakan Approximasi Butterworth dengan metode Transformasi Bilinear yang memenuhi spesifikasi yang diberikan.

1.3 Karakteristik Filter Butterworth

  • Maximally flat - respon magnitude sangat datar pada passband
  • Tidak ada ripple pada passband maupun stopband
  • Roll-off yang lebih lambat dibanding Chebyshev atau Elliptic
  • Karakteristik -3 dB pada frekuensi cutoff (sifat inheren)

1.4 Metode Transformasi Bilinear

Transformasi bilinear mengkonversi filter analog ke digital dengan hubungan:

s = (2/T) × (1 - z-1) / (1 + z-1)

Dengan prewarping untuk mengkompensasi frequency warping:

Ω = (2/T) × tan(ω/2)

3Parameter Band Pass Filter

Formula:

Ω₀ = √(Ωₗ × Ωᵤ)      [geometric mean]
B = Ωᵤ - Ωₗ      [bandwidth]
Perhitungan: Ω₀ = √(Ωₗ × Ωᵤ) = √(0.522787 × 0.605622) = √0.316611 = 0.562682 rad/s B = Ωᵤ - Ωₗ = 0.605622 - 0.522787 = 0.082834 rad/s Verifikasi: Ω₀² = Ωₗ × Ωᵤ 0.316611 ≈ 0.316611 ✓

4Normalisasi ke LPF Prototype

Formula:

A = |Ω₁² - Ω₀²| / (Ω₁ × B)
B_norm = |Ω₂² - Ω₀²| / (Ω₂ × B)
Ωᵣ = min(A, B_norm)
Perhitungan A (dari stopband bawah): A = |Ω₁² - Ω₀²| / (Ω₁ × B) = |0.426536² - 0.562682²| / (0.426536 × 0.082834) = |0.181933 - 0.316611| / 0.035332 = 0.134678 / 0.035332 = 3.811816 Perhitungan B_norm (dari stopband atas): B_norm = |Ω₂² - Ω₀²| / (Ω₂ × B) = |0.691143² - 0.562682²| / (0.691143 × 0.082834) = |0.477678 - 0.316611| / 0.057250 = 0.161067 / 0.057250 = 2.813382 Ωᵣ = min(3.811816, 2.813382) = 2.813382
Mengapa memilih minimum? Nilai minimum dipilih untuk memastikan kedua spesifikasi stopband (bawah dan atas) terpenuhi.

5Perhitungan Orde Filter

Formula Butterworth:

n = log₁₀(A²/ε²) / [2 × log₁₀(Ωᵣ)]

dimana: ε² = 10^(k₁/10) - 1   dan   A² = 10^(k₂/10) - 1
Perhitungan Parameter: ε² = 10^(k₁/10) - 1 = 10^(2/10) - 1 = 10^0.2 - 1 = 1.584893 - 1 = 0.584893 A² = 10^(k₂/10) - 1 = 10^(40/10) - 1 = 10^4 - 1 = 10000 - 1 = 9999.000000 Perhitungan Orde: n = log₁₀(A²/ε²) / [2 × log₁₀(Ωᵣ)] = log₁₀(9999/0.584893) / [2 × log₁₀(2.813382)] = log₁₀(17095.43) / [2 × 0.449229] = 4.232880 / 0.898457 = 4.711275 n (sebelum pembulatan) = 4.711275 n (setelah ceiling) = 5 Orde BPF aktual = 2n = 10

✓ Hasil Perhitungan Orde:

  • Orde LPF Prototype: n = 5
  • Orde BPF Aktual: 2n = 10

⚠ Catatan tentang -3 dB pada Cutoff:

Filter Butterworth secara matematis didefinisikan memiliki atenuasi tepat -3 dB pada frekuensi cutoff. Ini adalah karakteristik fundamental:

|H(jωc)| = 1/√2 ≈ 0.707 → 20×log₁₀(1/√2) = -3.01 dB

*Status PASS diberikan karena -3 dB adalah karakteristik standar Butterworth. Untuk ripple tepat -2 dB, gunakan filter Chebyshev Type I.
