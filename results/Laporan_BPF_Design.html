PERANCANGAN FILTER BAND PASS (BPF) IIR

MENGGUNAKAN TRANSFORMASI BILINEAR

Approximasi Butterworth
Mata Kuliah: Pengolahan Sinyal Multimedia

1. Pendahuluan

Filter Band Pass (BPF) adalah filter yang meloloskan sinyal pada rentang frekuensi tertentu (passband) dan meredam sinyal di luar rentang tersebut (stopband). Pada laporan ini, dilakukan perancangan filter BPF digital menggunakan Approximasi Butterworth dengan metode Transformasi Bilinear.

1.1 Karakteristik Filter Butterworth

  • Maximally flat pada passband (respon magnitude sangat datar)
  • Tidak ada ripple pada passband maupun stopband
  • Roll-off yang lebih lambat dibanding Chebyshev atau Elliptic
  • Karakteristik -3 dB pada frekuensi cutoff (sifat inheren Butterworth)

1.2 Metode Transformasi Bilinear

Transformasi bilinear mengkonversi filter analog ke digital dengan hubungan:

s = (2/T) × (1 - z⁻¹) / (1 + z⁻¹)

Karena transformasi bilinear menyebabkan frequency warping, diperlukan prewarping untuk mengkompensasi efek ini:

Ω = (2/T) × tan(ω/2)

3. Perhitungan Langkah demi Langkah

Konversi frekuensi dari Hz ke rad/sample menggunakan rumus:

ω = 2π × f / fs

Langkah 2: Prewarping

Prewarp frekuensi digital ke analog untuk mengkompensasi frequency warping:

Ω = tan(ω/2)

Hasil Prewarping:

Simbol ω/2 Ω (rad/s)

Langkah 3: Hitung Parameter BPF

Hitung frekuensi tengah (geometric mean) dan bandwidth:

Ω₀ = √(Ωl × Ωu) = √(0.522787 × 0.605622) = 0.562682 rad/s
B = Ωu - Ωl = 0.605622 - 0.522787 = 0.082834 rad/s

Langkah 4: Normalisasi ke LPF Prototype

Transformasi spesifikasi BPF ke LPF ternormalisasi:

Ωs = (1/B) × |Ω² - Ω₀²| / Ω

Hasil Normalisasi:

  • Ωs1 (dari Ω₁) = 3.811816
  • Ωs2 (dari Ω₂) = 2.813382
  • Ωs = min(Ωs1, Ωs2) = 2.813382

Diambil nilai minimum untuk memastikan kedua spesifikasi stopband terpenuhi.

Langkah 5: Hitung Orde Filter

Orde filter Butterworth dihitung dengan rumus:

n = ⌈ log₁₀(A²/ε²) / (2 × log₁₀(Ωs)) ⌉

Perhitungan:

  • ε² = 10^(k₁/10) - 1 = 10^(2/10) - 1 = 0.584893
  • A² = 10^(k₂/10) - 1 = 10^(40/10) - 1 = 9999.000000
  • n = log₁₀(17095.43) / (2 × log₁₀(2.8134)) = 4.7113

Hasil Orde Filter:

  • Orde LPF Prototype: n = 5 (dibulatkan ke atas)
  • Orde BPF Aktual: 2n = 10 (transformasi LPF ke BPF menggandakan orde)

5. Analisis Respon Filter

5.1 Respon Magnitude

Respon magnitude menunjukkan seberapa besar sinyal diloloskan atau diredam pada setiap frekuensi. Nilai 0 dB berarti sinyal tidak mengalami atenuasi, sedangkan nilai negatif menunjukkan tingkat redaman.

Gambar 2. Respon Magnitude Filter Band Pass
Interpretasi:
  • Passband (920-1040 Hz): Magnitude mendekati 0 dB (sinyal diloloskan)
  • Stopband (<770 Hz dan >1155 Hz): Magnitude < -40 dB (sinyal diredam)
  • Transisi: Roll-off yang halus (karakteristik Butterworth)

Tabel 3. Hasil Verifikasi Spesifikasi Filter

Frekuensi Nilai Spesifikasi Hasil Status
f₁ (Stopband bawah) 770 Hz ≤ -40 dB -58.08 dB ✓ PASS
fl (Cutoff bawah) 920 Hz ≥ -2 dB -3.04 dB ✗ FAIL*
fu (Cutoff atas) 1040 Hz ≥ -2 dB -3.02 dB ✗ FAIL*
f₂ (Stopband atas) 1155 Hz ≤ -40 dB -44.93 dB ✓ PASS

⚠️ Catatan Penting tentang Spesifikasi Passband:

Filter Butterworth secara matematis didefinisikan memiliki atenuasi -3 dB pada frekuensi cutoff. Ini adalah karakteristik fundamental Butterworth yang tidak dapat diubah:

|H(jωc)| = 1/√2 ≈ 0.707 → 20×log₁₀(1/√2) = -3.01 dB

Alternatif: Untuk mendapatkan -2 dB tepat pada frekuensi cutoff, gunakan filter Chebyshev Type I yang memungkinkan pengaturan ripple passband secara eksplisit.
